# Comprehensive Data-Driven Platform for Evaluating Abu Dhabi Strategic Initiatives

## Overview
This repository provides a solution for analyzing and evaluating data from the 'Evaluation of Joint Initiatives Aligned with Abu Dhabi Government's Strategic Objectives (2026)' dataset. The platform includes interactive dashboards and analytical tools to enable stakeholders to assess the alignment, impact, and performance of various initiatives.

## Features
- Interactive dashboards for data visualization.
- Tools for generating custom reports.
- Budget analysis (Planned vs Actual).
- Collaboration insights and impact assessment.
- Feedback mechanisms for stakeholder engagement.
- Integration with other datasets for a holistic view.

## Prerequisites
- Python 3.8 or higher
- Pandas
- Matplotlib
- Seaborn

## Installation
1. Clone the repository:
bash
$ git clone https://github.com/your-repo/abu-dhabi-strategic-initiatives.git

2. Navigate to the project directory:
bash
$ cd abu-dhabi-strategic-initiatives

3. Install the required Python libraries:
bash
$ pip install -r requirements.txt


## Dataset
Download the dataset file `2026 - Sub Awards Nomination Open Data.xlsx` and save it in the root directory of this project.

## Running the Examples
1. Open the `analysis_script.py` file.
2. Run the script using:
bash
$ python analysis_script.py

3. View the generated visualizations and summary report.
